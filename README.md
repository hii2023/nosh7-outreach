# NOSH7 Outreach — Batch Campaign Manager

An internal operations tool for the **NOSH7 Cloud Kitchen** team to manage outreach campaigns to subscription members at scale.

## What It Does

Allows the NOSH7 ops team to upload member data, track batch deliveries, update statuses in bulk, and send template-based messages — all from a single dashboard.

## Key Features

- **CSV / data upload zone** for importing member lists
- **Batch management table** with delivery filter and status tracking
- **KPI dashboard grid** — key outreach metrics at a glance
- **Quick filter bar** for fast segmentation
- **Bulk action bar** — update multiple records simultaneously
- **Status update modal** — change member status with a click
- **Template-based messaging** — consistent, reusable message formats
- **Rekart API integration** — connects to Rekart for message delivery
- **Rekart token management** — secure API token handling
- **Supabase backend** — real-time data storage and retrieval

## Tech Stack

- HTML, CSS, Vanilla JavaScript
- Supabase JS SDK
- Rekart API

## Audience

NOSH7 operations team managing subscription member outreach campaigns.
